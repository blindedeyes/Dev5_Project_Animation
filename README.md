All code is written by me, except code auto generated by visual studio projects.

# Camera controls:

w-forward

s- backward

a- left

d - right

space - up

x - down

# Debug controls:

return - draw bones

shift - draw wireframe

# ANIMATION CONTROLS:

1 (DEFAULT STATE) - Key cycling:
  Right Arrow- Next key
  Left Arrow- Prev Key
  
2 (Automatic state) - Plays animation for you, smoothly, with beginning and end interpolation
